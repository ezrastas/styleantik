Управление статичным содержимым

"Управление статичным содержимым" показывает список статичного содержимого вашего сайта.

Объект статичного содержимого - объект содержимого, не входящий в иерархию Раздел -> Категория -> Объект содержимого, т.е. является самостоятельным объектом.

Объекты статичного содержимого - это не часть раздела и не часть категории. Они не могут быть добавлены в блоги, в категории, в разделы. Единственный способ отображения статичного содержимого на сайте - это ссылка в меню.

Вы можете ввести свой текст в поле "Фильтр" для того, чтобы отсортировать список по своим критериям. Введите любой текст и нажмите "Enter". Также вы можете использовать поле "Сортировка", чтобы сортировать список по самым различным критериям.
 
Значки панели инструментов: 
Выберите нужный вам объект статичного содержимого или объекты статичного содержимого.
 
Новый: Создание нового объекта статичного содержимого.
 
Изменить: Редактирование уже существующего объекта статичного содержимого.

Допустить и Скрыть: Опубликование объекта на сайте и его скрытие.
 
В корзину: Отправка объекта статичного содержимого в корзину. Но он не будет полностью удален с вашего сайта. Вы сможете восстановить его на странице "Управление корзиной".


Описание столбцов:
Заголовок: Название объекта статичного содержимого. Нажатие на имя приводит к редактированию объекта.

На сайте:Показывает, опубликован объект на сайте или нет. Нажатие по иконке приводит к смене статуса.

Сортировка: Сортировка объектов статичного содержимого в списке.

Доступ: Уровень доступа к объекту статичного содержимого. Нажатие по ссылке приводит к смене уровня доступа.

ID: ID объекта, под которым он записан в базе данных.

Ссылок: Количество ссылок в меню на этот объект статичного содержимого.
 
Автор: Автор объекта.

Дата: Дата создания объекта статичного содержимого.
