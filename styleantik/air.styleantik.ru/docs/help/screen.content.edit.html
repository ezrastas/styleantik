Управление объектами содержимого: Создание / Изменение

Объект - отдельная часть содержимого в иерархии Joomla. Объектом содержимого может быть статья, категория содержимого, пункт меню или ссылка. Обычно, объекты содержимого выводятся в основной области сайта, называемой также main body. Объекты содержимого - самые последние в иерархии Разделов/Категорий, но также могут быть отдельными Объектами статичного содержимого.

Значки панели инструментов:

Просмотр: Открытие всплывающего окна, в котором можно просмотреть, как будет выглядеть объект содержимого на сайте.

Загрузка: Открытие нового окна, где вы сможете загрузить файл или изображение в папку images/stories сервера с вашего компьютера.

Сохранить: Сохранение раздела и возврат на страницу "Управление статичным содержимым".

Применить: Сохранение статичного содержимого без выхода со страницы редактирования. Удобно, когда вы собираетсь продолжить дальнейшее редактирование объекта.

Отмена: Отмена действия, при которой будут потеряны все изменения.


Информация о содержимом
 
Заголовок: Название статичного содержимого. Это название, под которым статичное содержимое будет показано на сайте.

Псевдоним заголовка: Это поле не используется ядром Joomla. Некоторый SEF-скрипты используют это поле для формирования SEF-ссылок.

Раздел: Выберите из списка раздел, которому будет принадлежать объект содержимого.

Категория: Выберите из списка категорию, которой будет принадлежать объект содержимого.

Вводный текст (обязательно): Введите здесь вступительный текст вашего объекта содержимого. Этот текст будет отображаться при просмотре блогов с участием этого объекта содержимого.

Основной текст (опционально): Основной текст вашего объекта содержимого. Он виден только при полном просмотре объекта и не отображается при отображении страницы в формате блог. 

Мамботы содержимого:
Мамботы содержимого - это специальные функции и скрипты, которые увеличивают простоту создания нового содержимого сайта. Эти мамботы будут доступны даже если в "Глобальной конфигурации" выбрано "Простой редактор HTML". Они могут быть опубликованы или отредактированы в "Управлении мамботами" (Мамботы -> Мамботы сайта).

По умолчанию включены два специальных мамбота содержимого:

  • {mosimage} (Мамбот mosimage): Используется для вставки изображений в содержимое.
  • {mospagebreak} (mospaging Joomla): Разбивка содержимого на несколько отдельных страниц.

Кнопки для вставки этих тэгов находятся под полем ввода текста (если они опубликованы) (Мамботы -> Мамботы сайта). Вы также можете просто вручную ввести код этих маботов в своем содержимом.

Форматирование содержимого
Опции форматирования содержимого сильно зависят от редактора, который вы используете на своем сайте.
Большинство редакторов - это свободно распространяемые редакторы с открытым исходным кодом (смотрите файловый архив http://developer.joomla.org и http://joom.ru) или коммерческие продукты.

По умолчанию Joomla имеет два встроенных редактора:

  • Простой редактор HTML: Этот редактор используется для ввода текста и его форматирования с помощью HTML-кодов.
  • TinyMCE: Это наиболее удобный редактор. Может работать в двух режимах: WISIWYG и HTML (смотрите более детальную помощь).

Для того, чтобы пользоваться редактором, необходимо, чтобы он был опубликован в "Управлении мамботами", а также был выбран как редактор по умолчанию в "Глобальной конфигурации".

Область атрибутов объекта содержимого состоит из пяти вкладок:
"Публикация", "Изображения", "Параметры", "Метаданные" и "Связь с меню".

Публикация
Информация о публикации

Опубликовано (на сайте): Позволяет опубликовывать объект содержимого на сайте.

Уровень доступа: Выберите уровень доступа к содержимому. Общий, Участники или Специальный.

Превдоним автора: Вы можете опубликовать содержимое под псевдонимом автора.

Изменить автора: Позволяет изменить автора содержимого на любого другого зарегистрированного пользователя.

Перезаписать дату создания: Позволяет перезаписать дату создания объекта содержимого. Вы можете изменить это в любой момент.

Начало публикации: Выберите дату, после которой содержимое автоматически будет опубликовано на сайте.

Окончание публикации: Если вы хотите прекратить показывать содержимое в какой-либо момент, то установите здесь дату, после которой объект содержимого будет скрыт на сайте.

ID: Показывает ID содержимого в базе данных.

Состояние: Показывает текущее состояние документа. Не редактируется.

Просмотров:Число просмотров этого объекта содержимого.
Кнопка 'Сбросить счетчик просмотров' сбрасывает число просмотров содержимого на ноль.

Версия: Показывает количество изменений документа.

Создано: Дата создания объекта.

Создал: Автор-создатель объекта.

Последнее изменение: Дата последней модификации документа.

Изменен: Имя пользователя, производившего изменения в документе.


Изображения
Управление изображениями MOS

Изображения галереи: Выберите из выпадающего списка изображение, которое хотите вставить в объект содержимого.
Отметьте нужное вам количество изображений и нажмите кнопку "Добавить".
Изображения появятся в списке "Изображения содержимого".

Изображения содержимого
Кнопками 'вверх' и 'вниз' вы можете перемещать изображения по списку и на странице. Изображения будут показаны относительно тэгов {mosimage} в вашем содержимом.

Вы можете задать выбранному изображению дополнительные параметры.
Выберите изображение в списке 'Изображения содержимого'.

Источник: Отображает имя выбранного изображения. Вы можете изменить имя.
Выравнивание: Выберите "Нет" для отмены выравнивание или выберите "По центру", "Слева" или "Справа".
Альтернативный текст: Введите здесь альтернативный текст изображения.
Рамка: Введите толщину рамки вокруг изображения (в пикселях).
Подпись: Подпись для изображения.
Выравнивание подписи: Выберите "Нет", "По центру", "Слева" или "Справа".
Ширина: Введите число пикселей ширины изображения.
Применить: Применение выбранных параметров для изображения.

Параметры
Управление параметрами

Суффикс CSS-класса страницы: Позволяет задать определенный стиль странице. С помощью этого можно достичь индивидуальности отдельных страниц сайта.

Кнопка Назад (Вернуться): Позволяет настроить, показывать или нет кнопку "Вернуться" внизу страницы и используемую для возврата на предыдущую страницу. Выберите 'Исп. ОБЩИЕ настройки' для использования параметров, заданных в "Глобальной конфигурации".

Название раздела: Показать/спрятать название раздела, к которому относится объект.

Название раздела как ссылка: Позволяет сделать названия разделов ссылками.

Название категории: Показать или спрятать название категории, к которой относится объект.

Название категории как ссылка: Позволяет сделать названия категорий ссылками.

Рейтинг объектов: Позволяет настроить, показывать или нет рейтинг объектов содержимого на сайте.

Имена авторов: Позволяет настроить, показывать или нет имена авторов при просмотре содержимого.

Имя автора: Позволяет показать/спрятать имена автором объектов содержимого.

Дата и время создания: Позволяет настроить, показывать или нет дату создания.

Дата и время изменения: Позволяет настроить, показывать или нет дату изменения содержимого.

Значок PDF: Позволяет настроить, показывать или нет значок PDF при просмотре содержимого.

Значок Печать: Позволяет настроить, показывать или нет значок печати документа.

Значок Email: Позволяет настроить, показывать или нет значок отправки документа по e-mail.

Ключевая ссылка: Текст ключа, по которому можно ссылаться на этот объект (например, в системе справки).

Тип Docbook: Экспериментальное поле, не для использования.

Метаданные
Данные мета-тегов

Описание: Это информация будет вставлена в header страницы с содержимым. Увеличивает индексируемость сайта.

Ключевые слова: Это информация будет вставлена в header страницы с содержимым. Увеличивает индексируемость сайта.

Добавить (Раздел-Категорию-Заголовок):  Позволяет добавить в поле ключевых слов страницы названия раздела, категории и названия содержимого.

Связь с меню

Позволяет создать 'Ссылка - Объект статичного содержимого' в выбранном меню.

Выберите меню: Выберите меню, в котором хотите создать ссылку.

Название пункта меню: Название пункта меню на объект содержимого, которое будет создано.

Связь с меню: Создание пункта меню.


Существующие связи с меню
