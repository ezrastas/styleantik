Расположение модулей

Страница расположения модулей позволяет вам редактировать/создавать позиции для модулей, которые вы впоследствии можете использовать в шаблоне.

Шаблоны определяют оформление вашего сайта Joomla.

Все стандартные позиции модулей уже введены и маловероятно, что вы будете их изменять. Зачем?

Если вы хотите создать новую позицию для модуля, вы просто можете ввести название позиции и дать ей описание. Описание давать необязательно.

Для того, чтобы использовать созданную позицию в шаблоне, вым нужно вставить следующий код в index.php вашего шаблон:.

<?php mosLoadModules ( 'position' ); ?>

Слово 'position' следует изменить на название позиции, которую вы создали.

После этого нажмите кнопку Предпросмотр сайта, чтобы узнать какие позиции для модулей используются в вашем шаблоне.
