Управление шаблонами

Этот раздел позволяет Вам назначать и редактировать шаблоны сайта.

Шаблоны определяют оформление вашего сайта Joomla.

Значки панели инструментов:

Выберите нужный вам шаблон.

По умолчанию: Делает выбранный шаблон шаблоном по умолчанию для всех пунктов меню/страниц.

Назначить: Открывает страницу, где Вы сможете назначить выбранный шаблон определенным пунктам меню/страницам. Этот шаблон будет использоваться на выбранных страницах вместо установленного по умолчанию. Если Вы выберите 'Отсутствует', это будет означать, что используется шаблон, установленный 'По умолчанию'.
Эта функция недоступна для шаблонов административной части.

Назначенный и установленный по умолчанию шаблон будет заменен шаблоном, выбранным в модуле сайта - Выбор шаблона, если он используется. Любой шаблон из списка будет доступен в модуле Выбор шаблона.

Новый: Позволяет загрузить/установить новый шаблон Вашего сайта.
 
Ред. HTML: Позволяет редактировать файл templates/выбранный-шаблон/index.php. Этот файл загружается, когда посетитель заходит на Ваш сайт.

Ред. CSS: Позволяет редактировать файл templates/выбранный-шаблон/css/template_css.css, который содержит CSS-стили, используемые в данном шаблоне
 
Удалить: Позволяет удалить шаблон. Вам будет предложено подтвердить операцию удаления. Выберите 'OK' чтобы удалить или 'Отмена' чтобы отменить операцию. Шаблон будет окончательно удален из базы данных.
 
Шаблоны административной части можно выбрать, щелкнув на значке "По умолчанию". Шаблоны административной части нельзя назначить отдельным страницам.
 

Описание столбцов:

Название: Это название шаблона. Если вы наведете указатель мыши на это название и флажок "Предпросмотр шаблона" будет установлен, Вы увидите уменьшенное изображение шаблона.
 
По умолчанию: Это шаблон, установленный как шаблон сайта по умолчанию в данный момент (только один шаблон может быть выбран как шаблон по умолчанию).
 
Назначен: Эта колонка показывает, является ли шаблон назначенным для какой-то страницы сайта.
 
Версия: Версия шаблона.
 
Дата: Дата создания шаблона.
 
URL автора: URL, предоставленный автором.
