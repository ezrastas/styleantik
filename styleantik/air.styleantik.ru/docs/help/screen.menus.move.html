Управление меню: Перемещение пункта

Эта страница позволяет перемещать пункты меню в друго существующее меню. Если Вы выбрали для перемещения пункт меню, содержащий подменю, то они также будут перемещены. 

Переместить в меню:
Выберите из списка меню, в которые вы хотите переместить пункт/пункты из другого меню.

Перемещаемые пункты меню:
Показывает список перемещаемых пунктов меню.

Значки панели инструментов:
Перенести: Перемещение пунктов меню.
