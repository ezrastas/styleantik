Управление меню / Главное меню и другие меню

Эта страница показывает список пунктов меню выбранного для просмотра.

Значки панели инструментов:

Новый: Создание нового пункта меню.

Изменить: Редактирование уже существующего пункта меню.

Допущено и Скрыто: Позволяют опубликовать или скрыть пункт меню. Вы можете одновременно опубликовывать или скрывать несколько пунктов меню.

Перенести: Перемещение существующего пункта из одного меню в другое(другие). Одновременно вы можете перемещать несколько пунктов.

Копия: Копирование существующего пункта меню. Вы можете копировать одновременно несколько пунктов меню.

В корзину: Перемещение пункта меню в "Корзину". Одновременно вы можете перемещать в корзину несколько пунктов. Пункты меню, находящиеся в корзине, могут быть обратно восстановлены в панели администратора: "Управление корзиной".

Отмена: Отмена сделанных изменений у пункта меню. Автоматически вы будете возвращены на предыдущую страницу.

Описание столбцов:
Пункт меню: Название ссылки в меню.  Нажмите на имя для редактирования пункта. Значок закмка рядом с названием пункта означает, что он "заблокирован".

На сайте: Показывает опубликован или нет пункт меню. Нажмите на значок для смены статуса.

Сортировка: Сортировка пунктов меню. Нажатия на иконки приводит к перемещению пункта меню вверх или вниз по списку.

Порядок: Порядок сортировки пунктов меню.

Доступ: Нажатие на ссылку приводит к смене прав доступа посетителей к пункту меню.

Itemid: Специфический "ID" (идентификационный номер) пункта меню в базе данных.

Тип: Тип пункта меню.

CID: Специфический номер пункта в базе данных.

Другие функции, доступные на этой странице:

Максимально уровней: Введите здесь число, отображающее максимальное число подменю у пункта.

Фильтр: Сортировка списка по вашим критериям.

Показывать по: Выберите число пунктов меню, которые будут показаны на одной странице.
