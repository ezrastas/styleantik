Управление разделами: Новый / Изменить

Раздел (Секция) - совокупность категорий, которые связаны между собой по смыслу.

Значки панели инструментов:

Загрузить: Дополнительная опция, с помощью которой вы можете загрузить файл или изображение в папку images/stories на вашем сервере. Удобно использовать, когда вам требуется, например, загрузить изображение для нового раздела.

Сохранить: Сохранение раздела и возврат на страницу "Управление разделами".

Применить: Сохранение раздела без возврата на предыдущую страницу. Удобно, когда вы собираетесь продолжить дальнейшее редактирование раздела.

Отмена: Отмена действия, при которой будут потеряны все изменения, сделанные в разделе.

Детали раздела

Заголовок: Название раздела, под которым он будет отображаться на сайте.

Раздел - Название: Название раздела, под которым он будет записан в базе данных и будет отображаться в панели администратора.

Изображение: Здесь вы можете выбрать изображение, которое будет показано на странице вместе с описанием раздела. Изображения в выпадающем списке находятся в папке images/stories вашего сервера.

Расположение изображения: Выберите расположение изображения в поле с описанием раздела. Слева или Справа.

Порядок отображения: Сортировка раздела среди других разделов. Эффект виден на странице "Управление разделами". Новые разделы по умолчанию добавляются в конец списка. Порядок сортировки может быть изменен только после сохранения объекта.

Уровень доступа: Уровень доступа посетителей сайта к разделу. Общий, Участники или Специальный.

Опубликован: Позволяет вам публиковать раздел на сайте или скрывать его.

Описание:  Описание раздела. Описание будет отображено под названием раздела при его просмотре.


Форматирование содержания
Опции форматирования содержимого сильно зависят от редактора, который вы используете на своем сайте.
Большинство редакторов - это открытые бесплатные редакторы (смотрите файловый архив http://developer.joomla.org и http://joom.ru) или коммерческие продукты.

По умолчанию Joomla имеет два встроенных редактора:
  • Простой редактор HTML: Этот редактор используется для ввода текста и его форматирования с помощью HTML-кодов.
  • TinyMCE: Это наиболее удобный редактор. Может работать в двух режимах: WISIWYG и HTML (смотрите более подробную помощь).

Для того чтобы пользоваться редактором, необходимо, чтобы он был опубликован в "Управлении мамботами", а также был выбран как редактор по умолчанию в "Глобальной конфигурации".

Вы не можете использовать тэг { mospagebreak } при вводе описания раздела.

Связь с меню

Эта опции позволит вам в несколько движений создать ссылку на раздел в меню.

Выберите меню: Выберите меню, в котором хотите создать ссылку на раздел.

Выберите тип меню: Здесь вы можете выбрать тип нового меню на раздел: 'Список раздела', 'Блог раздела' или 'Блог архива раздела'.

Название пункта меню: Название нового пункта меню, которое будет создано.

Создать пункт меню: Нажмите на кнопку для создания нового пункта меню.
