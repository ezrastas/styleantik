Управление языками

Эта страница показывает языки, которые установлены для Фронтэнда вашего сайта.

Значки панели управления:
Допустить:  Опубликование языка.

Новый: Нажмите "Новый" для загрузки и установки нового языка на ваш сайт.

Изменить: Нажмите на имя языка для редактирования деталей, или выберите язык и нажмите кнопку "Изменить".

Удалить: Удаление языка. Языковой файл будет полностью удален.

Помощь:  Нажмите "Помощь" для просмотра страницы с помощью.

Описание столбцов:

Язык: Название установленного языка.
   
На сайте: Показывает, опубликован ли язык или нет на сайте.
   
Версия: Версия языкового файла.
   
Дата: Дата языкового файла.

Автор: Автор языкового файла.
   
Email автора: Показывает email автора языкового файла.
