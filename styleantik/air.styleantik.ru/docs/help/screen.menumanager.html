Управление меню

"Меню" в Joomla являются группами навигационных ссылок на разделы, категории, элементы содержимого, компоненты или различные другие страницы. Эти ссылки называются "Пунктами меню".

Каждое меню должно иметь идентификационное имя, которое используется в системе данных Joomla.
Меню видно на сайте только тогда, когда опубликован модуль mod_mainmenu, содержащий это самое меню. Меню может иметь множество модулей mod_mainmenu.

Значки панели управления:

Новый - Создание нового меню. Появится окно "Информация о меню". "Имя меню" - Это идентификационное имя меню, используемое Joomla в своей системе данных. Имя должно быть уникальным. Рекомендуется не использовать в названии пробелы.

Дальше у нас идет "Заголовок модуля". Это имя нового модуля mod_mainmenu, автоматически созданного при сохранении меню. Это также имя модуля, которое появится на странице, если модуль опубликован. Имя может быть изменено при дальнейшем редактировании модуля. После того, как вы заполните эти поля, сохраните меню и оно появится в списке остальных меню Joomla.

Помощь: Меню пользователя

Помощь: Главное меню / Главная страница

Изменить - Редактирование меню.

Копия - Копирование меню. При этом будет создан дубликат копируемого меню. Вы можете скопировать любое меню.

Удалить - Удаление меню. Тогда откроется окно "Удалить меню : Название меню". Здесь показано, какой модуль и какие элементы меню будут удалены. Нажмите на иконку удаления, чтобы продолжить операцию. Вы должны будете подтвердить выполняемое удаление.

Удаленные меню не смогут быть восстановлены!

Описание столбцов:

Название меню - Нажмите на имя для редактирования меню.

Пункты меню - Нажмите на иконку для редактирования элементов меню.

Опубликовано - Число опубликованных элементов меню.

Скрыто - Число скрытых элементов меню.

В корзине - Число элементов меню, находящихся в корзине.

Модулей - Число модулей, присоединенных к этому пункту меню.
