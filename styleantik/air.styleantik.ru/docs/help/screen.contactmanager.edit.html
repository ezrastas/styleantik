Управление контактами: Новый / Изменение

Эта страница позволяет вам редактировать уже существующие контакты сайта или создавать новые.


Значки панели управления:

Сохранить: Нажатие на значке 'Сохранить' сохраняет измененный или вновь созданный контакт.

Закрыть: Отменяет сделанные изменения и выходит без сохранения контакта. После это происходит возврат в основное окно менеджера контактов. Контакт остается незаблокированным.

Детали контакта:
Любое пустое поле справа не будет отображено на экране при просмотре контакта.
Любое поле, которое является заполненным, автоматически добавляется в VCard контакта.

Категория: Выберите из выпадающего списка категорию, к которой будет относиться контакт.

Связано с пользователем: Выберите из списка любого пользователя, зарегистрированного на сайте, чтобы связать с ним текущий контакт.

Имя: Введите здесь имя контакта, которое будет отображено при просмотре. Вы также можете ввести имя пользователя, с которым связан этот контакт, или департамента, продукта, да что угодно.

Положение контакта: Лучше всего использовать совместно с именем пользователя, допустим, ввести его должность в компании. Например, "Руководитель".

E-mail: E-mail контакта, на который будут отправляться письма.

Также есть дополнительные 5 полей для отображения географической информации контакта.

  • Адрес (улица, дом)
  • Населенный пункт/Район 
  • Край/Область/Республика
  • Страна 
  • Почтовый индекс
Также присутствуют два поля для отображения телефонных номеров контакта.
  • Телефон 
  • Факс
Дополнительная информация:  Введите здесь любую информацию, которой вы хотите дополнить контакт.


Публикация:
Контакт сайта по умолчанию: Выберите "Да" или "Нет", чтобы сделать контакт контактом по умолчанию для вашего сайта.

Опубликован: Выберите "Да" или "Нет", чтобы показывать контакт посетителям сайта.

Порядок расположения: Сортировка контактов, если у вас на сайте их более одного.

Доступ: Выберите уровень доступа посетителей к контакту: Public, Registered или Special.

Изображения:
Здесь вы можете выбрать изображение (из папки images/stories) для отображения в контакте, или выберите "- Выберите изображение -", если не хотите отображать картинку в контакте.

Если изображение выбрано, вы увидете его в блоке предварительного просмотра.

Параметры:

Изображение меню: Маленькое изображение, располагаемое слева или справа от меню. Должно находиться в папке /images/stories.

Суффикс CSS-класса страницы: С помощью этого меню можно задать индивидуальный вид страницы контакта.

Значок Печать: Вы можете скрывать или показывать значок печати контакта. Вы также можете выбрать 'Исп. ОБЩИЕ Настройки', чтобы использовать настройки, которые заданы по умолчанию для сайта.

Кнопка Назад (Вернуться): Опция, с помощью которой вы можете включить или выключить ссылку "Назад", распологаемую внизу контакта и используемую для возврата на предыдущую страницу. Вы также можете выбрать 'Исп. ОБЩИЕ Настройки', чтобы использовать настройки, которые заданы по умолчанию для сайта.

Настройки для следующих 12 полей позволяют скрывать или показывать их при просмотре контакта:
  • Имя
  • Положение
  • E-mail
  • Адрес
  • Город/Пригород 
  • Край/Область/Республика
  • Страна 
  • Почта/Почтовый индекс
  • Телефон
  • Факс
  • Прочая информация
  • Изображение
Описание E-mail: Показать/Скрыть внизу контакта текст описания.

Текст описания: Текст описания для формы отправки e-mail.

Форма E-mail: Показать/Скрыть форму отправки e-mail.

Копия E-mail: Показать/Скрыть переключать отправки копии e-mail на адрес отправителя.

Раскрывающийся список: Показать/Скрыть раскрывающийся список при просмотре контакта.

Значки/текст: Показывать ли значки рядом с полями контакта или отображать только текст.

Если вы выбрали "Показывать значки", то для следующих полей контакта вы можете задать собственное изображение:
  • Значок адреса
  • Значок E-mail
  • Значок телефона
  • Значок факса
  • Значок прочего (дополнительной информации)
Если выбрано стандартное изображение, то оно должно находиться в папке  images/stories.
