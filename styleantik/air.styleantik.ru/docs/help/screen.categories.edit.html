Управление категориями: Новая / Изменить

Категорией называется совокупность объектов, которые связаны между собой по смыслу.

Эта страница позволяет вам создать новую категорию или изменить уже существующую.

Значки панели инструментов

Загрузить: С помощью этой функции вы сможете во всплывающем окне загрузить файл или изображение в папку images/stories.

Сохранить: Сохранение категории и возврат обратно в менеджер категорий.

Применить: Сохранение изменений без возврата в менеджер.

Отмена: Отмена изменений. Любые изменения будут потеряны. Если открыто окно редактирования категории, категория останется "незаблокированной".

Свойства категории

Заголовок категории: Введите здесь название категории, которое будет отображаться на сайте.

Название категории: Введите здесь название категории, которое будет отображено в панели администратора.

Раздел: Выберите из выпадающего списка раздел, которому будет принадлежать категория. Этот выбор требует применения изменений или сохранения.

Изображение: Выберите изображение, которое будет показано вместе с описанием категории. Изображения в выпадающем списке находятся в папке images/stories.

Расположение изображения: Выберите здесь расположение изображения по отношению к описанию категории. Слева или справа.

Порядок: Выберите здесь порядок отображения категории среди категорий в том же разделе. Это будет видно в "Управлении категориями".

Уровень доступа: Уровень доступа посетителей к категории. Общий, Участники или Специальный.

Опубликовано: Выберите, публиковать категорию на сайте или нет.

Описание: Введите здесь описание категории. Описание будет отображено при просмотре категории или в блоге.


Форматирование содержания
Опции форматирования содержания зависят от WISIWYG-редактора, который вы используете.
Большинство редакторов распространяются открыто и бесплатно (смотрите файловые архивы на http://forge.joomla.org и http://joom.ru/ ) или как платные продукты.

Joomla по умолчанию имеет 2 редактора:
  • Простой редактор HTML: обычный текстовый редактор, где вы можете вводить HTML-коды при форматировании содержимого.
  • Визуальный редактор TinyMCE: редактор, который может работать в режиме WYSIWYG или режиме HTML (Смотрите более подробную помощь).

Вы можете использовать редактор только тогда, когда он опубликован в "Управлении мамботами" и выбран как редактор по умолчанию в "Глобальной конфигурации".

Вы не можете использовать тэг { mospagebreak } в поле ввода описания.

Все изменения не будут потеряны только тогда, когда категория сохранена.

Связь с меню

Эта опция позволяет создать ссылку на категорию в меню.

Выберите меню: Выберите нужное вам меню (нажав на него), в котором будет создана ссылка.

Выберите тип меню: Здесь вы можете выбрать тип элемента меню, т.е. вид страницы, которая будет открываться после нажатия на этот пункт меню. Это могут быть 'Таблица содержимого категории', 'Блог содержимого категории' или 'Блог архивного содержимого категории'.

Название пункта меню: Название элемента меню, которое будет создано.

Связать с меню: Нажмите на эту кнопку, чтобы создать ссылку в меню.


Существующие связи с меню

Отображает ссылки на категорию уже существующие в меню.
