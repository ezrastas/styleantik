Управление шаблонами: Назначение

Шаблоны определяют оформление вашего сайта Joomla.

Эта страница позволит вам назначать различные шаблоны для определенных страниц, компонентов.

Выберите из списка слева меню, к которому/которым вы хотите назначить шаблон.
 
При выборе нескольких меню, они будут подсвечены соотвествующим образом.
После сохранения, для выбранных меню шаблон сайта по умолчанию будет отменен и заменен шаблоном, который вы им назначили.

Если вы выбрали 'Отсутствует', шаблон сайта по умолчанию будет назначен для всех  меню/страниц.

Панель управления:

Сохранить: Сохраняет ваш выбор и перемещает вас в менеджер шаблонов.

Отмена: Отменяет все сделанные вами действия и перемещает вас в менеджер шаблонов.
