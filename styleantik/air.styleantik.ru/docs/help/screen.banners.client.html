Управление клиентами баннеров

Эта страница показывает клиентов, баннеры которых могут быть показаны на вашем сайте.

Значки панели управления

Выберите клиентов, отмечая нужные вам.

Новый:  Создание нового клиента.

Изменить:  Изменение деталей клиента.

Удалить: Удаление клиента. Вы должны подтвердить операцию удаления. Нажмите "Да" для удаления клиента или "Нет" для отмены операции.

Описание столбцов

Имя клиента: Имя клиента.

Контакт: Отображение пользователя, отвечающего за этого клиента.

Количество активных баннеров: Отображает количество опубликованных баннеров на сайте этого клиента.
