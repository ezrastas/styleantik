Настройки экспорта новостей


Экспорт новостей позволяет вашему сайту экспортировать свои новости в специальных форматах, благодоря чему ваши новости можно быстро просмотреть, не заходя на сайт.

Кэш: Выберите "Да" или "Нет", чтобы включить или выключить кэширование лент новостей с вашего сайта.

Значение кэша, установленное в положение "Нет" повышает уязвимость вашего сайта, так как в таком режиме потенциальный взломщик может сильно загрузить сервер, вплоть до падения сервера.

Время кэширования: Время, через которое файлы кэша новостей будет обновляться на сервере. По умолчанию стоит 3600 секунд, или 60 минут. После того, как время прошло, Ленты новостей обновляют кэш на сервере.

Число объектов: Число объектов, доступных для экспорта и отображающихся в ленте новостей.

Заголовок: Заголовок экспортируемой ленты новостей с вашего сайта. Заголовок также является ссылок на ваш сайт.

Описание: Описание ленты новостей. Если ничего не введено, то отображается стандартное описание: "Экспорт с сайта Joomla!".

Изображение: Выберите изображение из выпажающего списка, чтобы вставить его в ленту новостей.

Альтернативный текст для изображения: Введите текст, который будет отображаться как альтернативный текст изображения.

Ограничение текста: Введите значение, которым нужно ограничить текст новостей в ленте новостей.

Длина текста: Длина текста в экспортируемых новостях ленты новостей.

При значение в "0" текст не будет показываться.

Порядок отображения: Порядок, в котором экспортируемые объекты будут отображаться.

  • 'По умолчанию'
  • 'Как на главной странице'
  • 'Сначала самые старые'
  • 'Сначала самые новые'
  • 'Заголовки по алфавиту'
  • 'Заголовки в обратном порядке алфавита'
  • 'Авторы по алфавиту'
  • 'Авторы в обратном порядке алфавита'
  • 'Сначала самые читаемые'
  • 'Сначала редко читаемые'

Живые закладки: Включить поддержку живых закладок для браузера Firefox.
