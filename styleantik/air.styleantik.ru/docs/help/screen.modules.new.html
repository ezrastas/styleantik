Управление модулями: Новый

Модули расширяют возможности Joomla и дают программам новую функциональность. Модули - небольшие объекты содержимого, которые могут быть отображены в любом месте шаблона, если это будет разрешено. Модули очень легко устанавливаются в Панели управления Joomla. Joomla уже включает в себя несколько модулей: Главное меню (Main Menu), Верхнее меню (Top menu), Выбор шаблона (Template Chooser), Опросы (Polls), Счетчик нажатий и т.д. Члены сообщества Joomla производят и распространяют модули Joomla сторонних производителей на общих основаниях. Они доступны для свободного скачивания на http://developer.joomla.org, joom.ru и многих других сайтах.

Пользовательские модули это модули, созданные, с помощью "Управления модулями" после нажатия на кнопку "Новый".
Вы можете создать модуль для отображения содержания или RSS-новостей с других сайтов.

Тип модуля: mod_user

Детали:
Заголовок: Название модуля, под которым он будет отображаться в фронтэнде и административной части вашего сайта.

Показывать заголовок: Возможность отключения и включения показа заголовка модуля в фронтэнде сайта.

Позиция: Выберите позицию, где будет отображаться модуль. Позиция должна поддерживаться вашим шаблоном.

Порядок модуля: Расположение модуля среди других модулей в выбранной позиции.

Уровень доступа: Уровень доступа посетителей к модулю. Общий (модуль виден любому посетителю), Участники (могут видеть только зарегистрированные пользователи), или Специальный (могут видеть только те пользователи, которые допущены к специальному уровню доступа).

Опубликован: Опции опубликования и скрытия модуля на сайте.

ID: Уникальный ID номер модуля, под которым он записан в базе данных.

Страницы / Пункты меню: Выберите страницы, на которых модуль будет отображен.


Параметры:

CSS-cуффикс класса модуля: Тип суффикса, используемого при отображении модуля (table.moduletable) для придания индивидуальных стилей каждому модулю.

Разрешить кэширование: Включение кэширования содержимого модуля.


Эта информация не касается импортирования новостей с помощью RSS:

Пользовательский код: Введите здесь любой текст или код для отображения в модуле на страницах вашего сайта.

Эта информация касается импортирования новостей с помощью RSS:

URL-адрес RSS: Введите адрес ссылки ленты новостей RSS/RDF.

Описание ленты: Показывает описание при просмотре ленты новостей.

Изображение ленты: Показывает изображение при просмотре ленты новостей.

Объектов: Количество отображаемых объектов в ленте новостей.

Описание объекта: Показывает вводный текст или описание некоторых объектов ленты.

Подсчет слов: Ограничение количества слов в ленте. При 0 показывается весь текст.
