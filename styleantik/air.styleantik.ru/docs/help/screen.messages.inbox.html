Личные сообщения


Эта страница отображает список принятых личных сообщений.

Для просмотра сообщения нажмите на название сообщения. Там же будут показаны детали сообщения. Нажмите кнопку 'Назад' для возврата в папку cообщений.
 
Для создания письма нажмите кнопку 'Новый'. Выберите получателя, введите тему письма и текст сообщения. Нажмите на кнопку 'Отправить' для отправки сообщения. Нажмите на кнопку 'Отмена' для отмены операции отправки.
 
Для удаления сообщения выберите нужное вам сообщение и нажмите на кнопку 'Удалить'.
