Управление меню: Копирование

Эта страница позволяет копировать меню и все его элементы.
Список показывает название копируемого меню и список копируемых пунктов меню.

Имя нового меню:
Введите здесь название нового меню.

Это идентификационное имя, которое используется Joomla в своей базе данных; оно должно быть уникальным.  Не рекомендуется использовать пробелы в названии меню.

Имя нового модуля:
Название новго модуля.

Это название нового модуля mod_mainmenu, который создается автоматически при сохранении нового меню. Также это имя отображается на странице, где опубликован модуль (имя можно сменить позднее при редактировании модуля).

Значки панели управления
Копия: Создание скопированного меню. Новое меню появится в списке "Управление меню", где позднее его можно будет отредактировать.
Также будет создан новый mod_mainmenu модуль.
Параметры созданного модуля могут быть отредактированы в 'Управление модулями': Модули -> Модули сайта.
