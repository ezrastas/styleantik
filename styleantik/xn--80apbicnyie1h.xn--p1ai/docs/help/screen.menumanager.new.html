Управление меню: Создание / Изменение

Имя меню:  Название меню.

Это идентификационное имя меню, используемое в базы данных системы Joomla; оно должно быть уникальным.  Не рекомендуется использовать пробелы в названии.

Сохранить:  Сохранение меню. Вы сможете увидеть его в списке меню.

Заголовок модуля: Название модуля.

Это имя нового модуля mod_mainmenu, создающегося автоматически при сохранения нового меню. Также это имя модуля отображается на странице, когда он опубликован (имя можно сменить позднее при редактировании модуля).

Сохранить:  Сохранение нового модуля mod_mainmenu.
Параметры созданного модуля можно изменить в 'Управление модулями': Модули -> Модули сайта
