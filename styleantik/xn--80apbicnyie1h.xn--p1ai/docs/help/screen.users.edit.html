Управление пользователями - Добавление или редактирование пользователя

Пользователи Joomla могут быть размещены в нескольких группах с различными уровнями доступа. Группы пользователей с доступом к фронтэнду включают Гостей (Anonymous), Пользователей (Registered), Авторов (Author), Редакторов (Editor) и Издателей (Publisher). Также имеется три группы с доступом к бэкэнду: Управляющие (Mamager), Администраторы (Administrator) и Главные администраторы (Super Administrator).

Этот экран позволяет вам создавать нового пользователя или редактировать уже существующего.

Панель управления:
Сохранить: Нажмите для сохранения создания/редактирования пользователя, после чего вы вернетесь в менеджер пользователей.

Применить: Сохранение создания/редактирования пользователя, но вы остаетесь на этой странице для дальнейшего редактирования.

Отмена: Отмена всех сделанных изменений, пользователь остается незаблокированным.

Информация о пользователе:
Полное имя: Здесь вводится реальное имя пользователя, под которым он будет отображен в менеджере пользователей.

Имя пользователя: Здесь вводится Ник пользователя, под которым он будет входить в систему сайта.

Email: Почтовый адрес пользователя.

Новый пароль: Пароль пользователя.

Проверка пароля: Подтверждение пароля пользователя.

Группа: Выберите тип пользователя.

Блокировать пользователя: Позволить пользователю входить в сайт или нет.

Получать системные сообщения на e-mail: Включение или отключение возможности для администратора посылать уведомления на e-mail о неполадках на сайте.

Дата регистрации: Отображает дату, когда пользователь был зарегистрирован на сайте.

Дата последнего посещения: Отображает дату, когда пользователь последний раз авторизировался на сайте.


Контактная информация:
Отображает, назначен ли пользователю какой-либо контакт сайта или нет (Компоненты -> Контакты -> Управление контактами).
