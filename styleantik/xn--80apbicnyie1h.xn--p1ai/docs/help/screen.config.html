Глобальная конфигурация

Глобальная конфигурация управляет основными настройками Joomla. Изменения, сделанные в этой области панели администратора, также, соответственно, изменят файл configuration.php, содержащий ваши настройки. Для того чтобы отредактировать какие-либо настройки вашего сайта, этот файл должен быть перезаписываемым (настраивается установкой CHMOD).

Справа от информации о правах этого файла находится опция защиты этого файла от записи - "Защитить от записи после сохранения". Выбор этой опции поставит на конфигурационный файл Joomla запрет перезаписывать его. Если ваш сайт уже настроен, как вам хочется, эта опция - прекрасная возможность повысить безопасность сайта.


Сайт:

Сайт выключен - С помощью этого параметра Вы можете эффективно включать и выключать 'Фронтэнд'.

Фронтэнд - часть Web-сайта, которая видна посетителям (пользователям).

Используйте эту опцию, если вы собираетесь устанавливать новые модули или обновлять данные, и не хотите, чтобы пользователи видели это, пока работа не будет закончена. Не забудьте выключить эту опцию, установив значение "Нет", после завершения обновления сайта.

Сообщение при выключенном сайте - Эта опция полезна, когда ваш сайт находится в выключенном режиме. Тогда на экране браузера будет отображаться сообщение, которое вы написали. Сообщение по умолчанию:  "Сайт недоступен. Пожалуйста, зайдите позже".
Вы можете изменять этот текст на любой, какой вам нравится.

Сообщение при системной ошибке - Это сообщение выводится, если подключение к БД не было завершено успешно.  Так же, как и с предыдущей опцией, вы можете вписать любой текст. Разработчики ввели возможность использовать стандартные HTML-тэги в этом окне, например, <br /> для перехода на следующую строку. Вы также, например, можете добавить сюда адрес электронной почты (e-mail) для того, чтобы посетители сайта могли связаться с вашим администратором или вебмастером.

Название сайта - Эта опция позволит вам установить название вашего сайта, которое будет отображено в заголовке содержимого сайта. Это также может быть отображено в шаблоне, для чего нужно вставить следующий код в файл шаблона [Joomla_root/templates/template name/index.php]:

%%<?php echo $_CONFIG->SITENAME; ?>%%

Показывать ссылки неавторизованным - С этой включенной опцией все посетители сайта  будут видеть то содержание, к которому они имеют доступ, (это уровень доступа 'Пользователям' или 'Специальный'). С выключенной опцией содержание будут видеть посетители, относящиеся к группе 'Посетители'.  Уровень доступа выбирается при создании содержимого.

Разрешить регистрацию пользователей - Опция позволяет посетителям сайта регистрироваться у вас и заводить свои аккаунты. Если выбрано "Да", форма авторизации будет включать в себя подсказку "Не зарегистрированы? Регистрация". Если выбрано "Нет", эта надпись исчезнет с формы авторизации.

Активация нового аккаунта - Если выбрано "Да", каждому пользователю при регистрации на e-mail будет выслана ссылка для подтверждения регистрации. Пользователи не будут зарегистрированы на сайте, пока не пройдут по ссылке активации.  Если выбрано "Нет", пользователи будут зарегистрированы без активации.

Требовать уникальный е-mail - Выберите эту функцию, чтобы предотвратить создание аккаунтов с одним адресом e-mail.

Режим отладки - Опция позволяет вам видеть сообщения об ошибках при выполнении каких-либо php-сценариев сайта. Это отличный способ наиболее полно сообщить о вашей ошибке администратору или вебмастеру.

WYSIWYG-редактор - Выберите один из визуальных редакторов, установленных у вас на сайте, для редактирования содержимого.

Длина списка - Выберите длину списков по умолчанию в панели администратора (Бэк-энд).

Значок сайта в Закладках - Опция позволяет указать значок, который будет отображаться в браузере рядом с названием или адресом вашего сайта, а так же в Закладках.  По умолчанию используется эмблема Joomla.  Если у вас имеется другой значок, то Вы должны закачать его на сервер и заменить им стандартный значок.


Локаль:

Язык -  Выберите язык, который будет использоваться по умолчанию на вашем сайте.  Если это другой язык, отличный от английского, сначала вы должны установить его с помощью инсталлятора Joomla в панели администратора.

Часовой пояс - Опция позволяет установить смещение времени вашего сервера относительно GMT (Время Гринвича - UK). Выберите смещение из выпадающего списка. 

Локаль страны - Региональные настройки страны вашего веб-сайта.  Например, ru_RU - это настройка локали для России.
Установка правильной локали влияет на правильное отображение даты, времени и т.д. По умолчанию установлена локаль для русского языка - ru_RU.cp1251. Но это может отличаться от локали на вашем сервере. Уточните подробности у своего провайдера. Пользователям серверов на базе MS Windows необходимо будет установить локаль RU.


Содержимое:

Заголовки в виде ссылок - Выберите 'Да', если вы хотите, чтобы названия новостей были ссылками, которые будут ввести в полный текст содержимого.  Выберите 'Нет', и ссылка 'Подробнее...' будет на полный текст содержимого.

Включить "Далее..." как ссылку - Выберите "Показать" для отображения ссылки "Подробнее" в конце "вступительного текста" содержания.

Рейтинг/Голосование - Опция позволяет оценивать содержимое сайта.  Выберите 'Да', и 'Панель звезд' будет показываться в каждом содержимом, где вы можете оценить текст.

Форма оценки не показывается в "Блогах" содержимого, и она показывается только для того содержимого, который имеет вступительный текст.

Имена авторов - Выберите "Показать" для отображения имени автора в содержимом.

Дата и время создания - Выберите "Показать" для отображения даты создания содержимого.

Дата и время изменения - Выберите "Показать" для отображения даты изменения изначального текста содержимого.

Кол-во просмотров - Выберите "Показать" для отображения счетчика просмотров статьи/содержимого.

Значок PDF - Выберите "Показать" для отображения значка PDF в содержимом сайта.

Эта опция не будет работать, если ваша папка media неперезаписываемая.

Значок Печать - Выберите "Показать" для отображения значка печати содержимого сайта.

Значок E-mail -  Выберите "Показать" для отображения значка "Отправить..." для содержимого сайта.

Значки - Выберите "Показать" если вы хотите, чтобы ссылки E-mail/Печать/PDF показывались в виде значков.

Таблица с оглавлением для многостраничных объектов - Выберите "Показать", чтобы отображать навигацию по содержанию,  разбитому на несколько страниц. Для создания новой страницы в содержимом, вы должны использовать мамбот разбиения на страницы - {mospagebreak}, который можно найти в любом редакторе.

Кнопка Назад (Вернуться) -  Выберите "Показать"  для отображения кнопки "Назад" (Вернуться), что облегчает навигацию по вашему сайту.

Навигация по содержимому - Выберите "Показать" для опубликования ссылок  'Следующий' и 'Предыдущий' при просмотре списка категорий или  для изменения страницы при просмотре статьи.

Многоязыковая поддержка - позволяет просматривать сайт на разных языках. Для этого должен быть установлен компонент MambelFish.


База данных:

Это очень важные настройки сайта. Если ваш сайт на Joomla установлен и работает без перебоев, крайне не рекомендуется изменять эти настройки!

Имя хоста - Например, 'localhost' - типичное название хоста. Более точно нужно узнавать у провайдера.

Имя пользователя MySQL - Введите имя того пользователя, который имеет доступ к БД Joomla.

Пароль для доступа к MySQL -  Введите пароль, который требуется для подключения к БД Joomla.

База данных MySQL - Введите название БД, куда были установлены таблицы Joomla.

Префикс базы данных MySQL - Введите префикс таблиц, которые были установлены при установке Joomla. По умолчанию префикс таблиц - "jos_".

НЕ ИЗМЕНЯЙТЕ, ЕСЛИ ВЫ УЖЕ ИМЕЕТЕ НАСТРОЕННЫЙ САЙТ, ТАБЛИЦЫ КОТОРОГО УЖЕ ИМЕЮТ ПРЕФИКС, КОТОРЫЙ УКАЗАН У ВАС В НАСТРОЙКАХ!!!

Сервер:

Элементы, которые были установлены в процессе установки сайта, и не могут быть изменены через эту вкладку.

Абсолютный путь -  Информация о том, куда была установлена Joomla на вашем сервере.

URL сайта - Адрес вашего веб-сайта.

Секретное слово - Уникальная комбинация символов, которая генерируется при установке Joomla. Используется для повышения безопасности.

GZIP-сжатие страниц - Выберите "Да", для того, чтобы перед отправкой, сервер сжимал данные, конечно, если это поддерживается самим сервером. Это позволяет уменьшить время загрузки страниц при просмотре сайта.

Время жизни сессии - Введите любое время (в секундах) жизни сессии пользователей панели управления Joomla!, после чего будет произведен автоматический выход из системы при бездействии пользователя.

Если вы редактировали текст содержимого, но не успели его сохранить в пределах означенного времени, то все изменения пропадут.

Сообщения об ошибках - Если вы используете опцию "Режим отладки", то выберите из выпадающего списка уровень отображения ошибок, который будет отображаться.
Возможные варианты: Настройки системы, Нет, Простые и Максимум (Все).

  • Настройки системы -  Это позволяет показывать системные ошибки при выполнении php-сценариев.
  • Нет - Отключает сообщения об ошибках.
  • Простые -  Отображает ошибки при выполнении простых сценариев, используется значение PHP error_reporting (E_ERROR|E_WARNING|E_PARSE)
  • Максимум (Все) -  Максимальный уровень отображения ошибок, используемые опции PHP: error_reporting (E_ALL)

Создание файлов - Значение прав, которые устанавливаются по умолчанию при установке компонентов, модулей и мамботов. Выберите "Установить CHMOD для новых файлов", где вы можете установить свои права.

Создание каталогов - То же самое, что и при создании новых файлов, только здесь права ставятся на новые папки. Выберите "Установить CHMOD для новых каталогов" для установки своих прав.


Метаданные:

Описание сайта, которое индексируется поисковиками -  Описание сайта (мета-тэг description), которое будет показано в поисковых серверах. Вы можете ограничить ваше описание, например 20-ю символами, в зависимости от поискового сервера. Описание лучше всего давать кратким и полностью подходящим под ваш сайт. Также, в описании вы можете использовать ваши ключевые слова и фразы. Так как некоторые поисковые сервера читают больше 200 символов, то вы можете написать одно-два предложения. Также удостоверьтесь, что наиболее важная часть вашего описания находится в первых 200 символах. Этот тэг является тэгом по умолчанию. К нему добавляется содержимое тэга объекта содержимого, если оно заполнено.

Ключевые слова сайта - Оптимальное число ключевых слов (мета-тэг keywords) вашего сайта должно составлять от 300 до 500 символов (рекомендуется не более 200 символов). Не повторяйте слова, а также отделяйте друг от друга запятой или пробелом. Например, 'Русский дом Joomla'. Этот тэг является тэгом по умолчанию. К нему добавляется содержимое тэга объекта содержимого, если оно заполнено.

Показывать мета-тэг title - Показывать название содержимого в html-коде при просмотре сайта.

Показывать мета-тэг author - Показывать имя автора содержимого в html-соде при просмотре сайта.

Если вы хотите, чтобы ваш сайт находился на высоких позициях в рейтинге поисковых сервером, вы должны как минимум изучить каждый поисковый сервер, чтобы наиболее полно удовлетворить условиям индексации сайта.


Почта:

Выберите, какую функцию отправки почты вы хотите использовать: Функцию PHP-mail, Sendmail или SMTP-сервер.

Письма от - Выберите адрес, который будет отображаться в поле письма "От кого".

Отправитель - Выберите и напишите здесь имя отправителя письма.

Путь к Sendmail - абсолютный путь к программе Sendmail.

Использовать SMTP-авторизацию - Выберите "Да", если на вашем сервере для отправки почты необходима smtp-авторизация.

Имя пользователя SMTP - Если у вас включена опция авторизации, то введите здесь имя пользователя.

Пароль для доступа к SMTP - Если у вас включена опция авторизации, то введите здесь пароль пользователя.

Адрес SMTP-сервера - Введите название smtp-сервера, используемого для отправки писем, например, localhost.


Кэш:

Использовать кэширование - Выберите "Да", чтобы включить кэширование содержимого Joomla. Это позволяет повысить производительность вашего сайта.

Папка кэша - Папка для хранения файлов кэша сайта. Пака должна быть перезаписываемой.

Время жиэни кэша - Отредактируйте максимальное время жизни кэша (в секундах), по истечении которого он будет обновлен.


Статистика:

Использовать сбор статистики - Выберите "Да", чтобы собирать информацию о посетителях и их браузерах, а также историю просмотра содержимого. Если выбрано "Нет", статистика не будет собрана.

Вести статистику просмотра содержимого по дате - Выберите "Да", чтобы собирать только историю просмотра содержимого.

В этом режиме записываются очень большие объемы данных!

Статистика поисковых запросов - Выберите "Да" для включения статистики поисковых запросов пользователей.


SEO:

Дружественные для поисковых систем URL-адреса -   Изменяет общий вид ссылок и оптимизирует их для того, чтобы увеличить поисковую индексацию сайта. Опция работает ТОЛЬКО на тех серверах, где установлен веб-сервер Apache и подключен его модуль mod_rewrite.  Выберите "Да" для включения режима дружественных ссылок. Для того чтобы режим SEF работал нормально, необходимо также переименовать файл htaccess.txt в .htaccess.

Динамические заголовки страниц -  Выберите "Да", чтобы включить динамическое изменение заголовков страниц, в зависимости от просматриваемого содержимого. Опять же, это сделано для улучшения индексации сайта.
