Поисковые запросы


Эта страница показывает сохраненные логи поисковых запросов ваших посетителей. Список содержит:

Текст поиска: Показывает фразу, введенную посетителем при поиске.
 
Запросов: Показывает число запросов с похожим текстом при поиске.
 
Возвращено результатов: Показывает число страниц, выданных при поиске посетителю.
 
Некоторые поисковые фразы иногда не совпадают с материалом на вашем сайте и поэтому посетителю не выдаются положительные результаты. С помощью данного сервиса, вы можете оценить насколько ваш материал соответствует ключевым словам вашего сайта.

Для того, чтобы этот сервис работал, вы должны его включить: Опция "Статистика поисковых запросов" в "Статистика" глобальной конфигурации сайта.
