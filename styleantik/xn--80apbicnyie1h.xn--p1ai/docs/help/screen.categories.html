Управление категориями


Категорией называется совокупность объектов, которые связаны между собой по смыслу.

Эта страница показывает категории вашего сайта. Каждая категория принадлежит определенному разделу. Выбор раздела из выпадающего списка позволит вам отображать категории, принадлежащие только этому разделу.

Так вы можете фильтровать категории с помощью разделов.

Значки панели управления:
Выберите категории, отмечая нужные вам.

Допустить и Скрыть: Функция публикации позволяет опубликовать категории на сайте или скрыть их. Эту функцию вы можете применить сразу к нескольким категориям.

Новый: Создание новой категории.

Изменить: Редактирование уже существующей категории.

Переместить: Откроется страница "Перемещение категорий", где вы можете перенести категории со всем содержимым из одного раздела в другой.

Копия: Откроется "Копирование категорий", где вы сможете скопировать категории со всем содержимым в другой раздел.

Удалить: Удаление категории. Вы должны подтвердить выполнение этого действия.


Описание столбцов:

Название категории: Имя категории. Нажмите на имя для перехода к редактированию категории. Значок замка рядом с категорией показывает, что категория "заблокирована".

На сайте: Показывает, опубликована категория или нет. Нажмите на иконку для смены статуса.

Порядок: Порядок сортировки категорий в списке (на сайте не отображается). Каждой категории вы можете вручную присвоить свой порядковый номер и нажать на значок "Сохранить порядок" для пересортировки по вашим параметрам.

Доступ: Показывает, для какого типа посетителей эта категория: Public, Registered или Special.

Раздел: Название раздела, которому принадлежит категория. Нажмите на имя для перехода к редактированию раздела.

ID категории: Отображает идентификационный номер (ID) категории, под которым она записана в базе данных.

В архиве: Число элементов содержимого в категории.

В корзине: Число элементов содержимого категории, перемещенных в корзину.
