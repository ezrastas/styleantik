Управление баннерами: Новый / Изменить

--- Предварительная версия

Эта страница позволит вам создать новый баннер или отредактировать уже существующий.

Значки панели управления:

Загрузить: Откроется всплывающее окно, где вы сможете загрузить файл или изображение баннера в папку images/banners с вашего компьютера.

Сохранить: Сохранение изменений и возврат в менеджер баннеров.

Закрыть: Отмена изменений. Любые изменения будут потеряны. Баннер останется "незаблокированным".


Подробности баннера:

Название баннера: Имя баннера.

Имя клиента: Выберите клиента, которому будет принадлежать баннер.
 
Сколько раз показать: Введите число показов баннера на вашем сайте, или выберите "Всегда показывать" для бесконечного показа баннера.
 
Показывать баннер: Выберите "Да" или "Нет".
 
URL клика: Введите здесь адрес URL, на который пользователи будут попадать при нажатии на баннер.
 
Нажатий: Кнопка "Сбросить число нажатий": сбросит число нажатий на баннер и установит значение равное 0.

Изображение баннера: Показывает изображение, которое вы выбрали для баннера.
